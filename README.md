# Resume Analyzer AI

AI-powered resume analyzer built using LangChain, Groq, ChromaDB, and Streamlit.

## Features
- Resume analysis
- Skill gap detection
- ATS-style feedback
- AI-powered Q&A

## Technologies Used
- Python
- Streamlit
- LangChain
- Groq
- ChromaDB

## Run Locally

pip install -r requirements.txt

streamlit run app.py
